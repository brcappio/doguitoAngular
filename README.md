# Doguito Angular

Doguito is a petshop system, based on forms and CRUD.
Ported and improved an old project as part of my Angular learning.

With this project I learned about:
- PrimeNG
- PrimeFlex
- JSONServer
- APIs
- FormBuilder and NonNullableFormBuilder
- Form Validators 
- Masks
- CRUD
- More on routing
- Imports
- Models
- Styling
